# Todo App

A CLI-based Todo application built using Spec-Driven Development.

## Features
- Add Task
- View Tasks (status, title, description)
- Update Task
- Delete Task
- Mark Task Complete/Incomplete

## Requirements
- Python 3.13+
- UV (Python package manager)

## Setup
1. Ensure you have `uv` installed.
2. Clone the repository.

## Usage
Run the application using:
```bash
python main.py
```

## Spec-Driven Development
This project follows strict SDD principles. Implementation is generated based on specifications in the `/specs` directory.
